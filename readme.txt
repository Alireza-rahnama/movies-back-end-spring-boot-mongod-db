This is a full stack java application that features implementation of Java Spring Boot, server side coonection to mongodb,
 and REST Api, with a loosely coupled and Multi layered architecture(Model, controller, view.)

Java-Compiler and Runtime: Openjdk@17
-MangoDb
-React.js
-springboot@3.0.6
-IntelliJ IDEA 2023.1 (Ultimate Edition)

1.run the application
2.navigate to http://localhost:8080/api/movies to retrive all movies available in the database
2.navigate to http://localhost:8080/api/movies/{imdbID} to retrive a single movie
